# MitiSync Suite

**Live Demo:** [https://mitisync.vercel.app](https://mitisync.vercel.app)

Align your timelines and bridge your languages in one minimal workspace. MitiSync Suite is a collection of high-performance, lightweight tools wrapped in a clean, modern aesthetic.

## Features

- **Patro Sync:** Synchronize your dates. Accurate AD to BS conversions with full leap-year support and an interactive Nepali calendar picker.
- **Bhasa Sync:** Bridge the language gap with lightning-fast, side-by-side English to Nepali translation powered by Google Translate API.

## Tech Stack

- React 18 + Vite
- Tailwind CSS v4
- TypeScript
- React Router v6
- React Datepicker & Nepali Datepicker

## Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
